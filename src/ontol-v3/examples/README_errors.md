# Примеры TDL с ошибками (для демонстрации валидации)

Используйте эти файлы во вкладке **«Рендер TDL»** в Streamlit, чтобы показать, как приложение сообщает об ошибках.

| Файл | Проверка | Ожидаемое сообщение |
|------|----------|---------------------|
| `error_cycle.tdl` | Цикл наследования | «Обнаружен цикл наследования с участием '…'» |
| `error_multiple_inheritance.tdl` | Конфликт множественного наследования | «Конфликт множественного наследования в 'Дитя': атрибут 'x' имеет разные типы от родителей …» |
| `error_unknown_class_assoc.tdl` | Неизвестный класс в ассоциации | «Участник ассоциации не найден: … или Призрак» |
| `error_unknown_type_attr.tdl` | Неизвестный тип атрибута | «Атрибут '…' класса '…' ссылается на неизвестный тип 'Призрак'» |
| `error_unknown_general.tdl` | Несуществующий родитель в обобщении | «Unknown classifier 'НесуществующийРодитель'» |

В каждом файле в начале (комментарий `--`) кратко описана ошибка и ожидаемый результат.
